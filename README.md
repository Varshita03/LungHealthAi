# 🫁 LungHealthAI – Respiratory Disorder Detection

An AI-powered system that detects respiratory diseases such as Asthma, Pneumonia, and COPD using lung sound analysis.

---

## 🚀 Features

* Upload lung sound (.wav) files
* Detect respiratory diseases using CNN model
* Audio feature extraction using Librosa (MFCC, Mel Spectrogram, Chroma)
* Web interface for easy interaction

---

## 🧠 Tech Stack

* Python
* Flask
* TensorFlow / Keras (CNN)
* Librosa (Audio Processing)
* HTML, CSS

---

## 📂 Project Structure

* `backend/` – Flask app and model integration
* `templates/` – Frontend UI
* `static/` – CSS and uploads
* `models/` – Trained models (.h5)

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```

---

## 📊 Results

* Achieved accurate classification of respiratory diseases using CNN model
* Successfully processed lung audio signals into meaningful features (MFCC, Mel Spectrogram)
